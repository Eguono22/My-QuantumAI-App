### Suggested Scoring Guide

Trust score: `4` if the user now trusts the workflow enough for tiny paper trades, `5` if they say the proof is strong enough to use repeatedly.

Value score: `4` or `5` if the user says it saves time, improves discipline, or makes trade review more repeatable.

Would pay: `Maybe` for cautious interest, `Yes` only if they name a concrete paid outcome.

Common friction to listen for: `Still wants more repeated history, broader track record, or stronger performance proof.`
